About this Extension

Hey there! I built this extension once when I was thinking there was no good way to preview the awesome themes provided by Bootswatch, outside of their bookmarklet. To create something more visual and fun to use, I built Bootswatch Previewer. With a Material Design inspired layout and experience, this extension gives you an awesome way to try out all the themes available.
